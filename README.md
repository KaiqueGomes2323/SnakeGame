PROJECT TITLE: Snake Game
PURPOSE OF PROJECT: O jogo foi criado com o objetivo de ser um minigame competitivo entre amigos no mesmo computador, foi feito para ser leve e rodar em qualquer máquina.
VERSION or DATE: Versão 2.1.0 - 13/03/2026
HOW TO START THIS PROJECT: Para iniciar o projeto basta baixar esse repositório e ter o Greenfoot instalado no seu computador, após baixar esses dois, vá na pasta do repositório instalado e abra o aquivo chamado "project" ou "projeto".
